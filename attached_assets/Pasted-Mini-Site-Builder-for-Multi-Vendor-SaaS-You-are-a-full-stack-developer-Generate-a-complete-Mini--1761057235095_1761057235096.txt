Mini-Site Builder for Multi-Vendor SaaS
You are a full-stack developer. Generate a complete **Mini-Site Builder** feature for a multivendor SaaS marketplace, where every vendor can create a fully functional mini-website under a subdomain (e.g., businessname.myapp.in) with a slot-based no-code wizard.  

The system should support the following features:

1️⃣ **Multi-Tenant Architecture**
- One codebase serving all vendors.
- Each subdomain maps dynamically to vendor data.
- Dynamic theme engine for each vendor.
- Wildcard subdomain support.

2️⃣ **Slot-Based Website Builder Wizard**
- Vendors fill a step-by-step form (slots) to create their website.
- Sections/slots:
    - Business Info: name, tagline, category, subcategory, about, logo, cover image(multiple), owner name(multiple adding option), year founded.
    - Location & Contact: address, city, state, pincode, Google Map pin, phone, WhatsApp, email, working hours, holidays, emergency availability.
    - Branding & Design: theme template selection, primary/secondary color, font, hero layout, gallery, CTA buttons, social links.
    - Catalog / Products / Services:vendor can select the products and services from their catalogue, or create a custom catalogue but it needs admin aprooval
    - our team : add employees,  roles, , profile images.
    - Publish & SEO: preview site, set subdomain,SEO meta, enable/disable chat buttons.
- Live preview updates as vendors fill slots.
- Progress bar indicating completion %.
-a form for lead generation

3️⃣ **Database Schema**
- Multi-tenant structure storing all vendor website data in JSON format.
- Support for master catalogue, custom listings, employees, bookings, and product/service data.
- Include example collections/tables for:
    - Business profile
    - Contact info
    - Branding/theme
    - Catalog/services/products
    - Employees(team)
    - Slots/progress
    - Published site status

4️⃣ **Frontend**
- Built with React or Next.js.
- Slot wizard with step-by-step navigation.
- Live preview pane for instant visualization.
- Theme engine applies colors, fonts, layout, and buttons dynamically.
- Form validation for required fields and correct formats.
- Responsive, mobile-first design.

5️⃣ **Backend**
- Node.js + Express API.
- Endpoints for:
    - Save slot data
    - Publish website
    - Fetch website data by subdomain
    - Admin approval for custom listings
    - List available themes and master catalog
- Handle multi-tenant subdomain routing.
- Store all uploaded media (logo, gallery, banner) securely.
- Include authentication (vendor login, admin roles).

6️⃣ **Automatic Actions**
- Generate subdomain on publish (businessname.myapp.in)
- Dynamic website rendering based on JSON data
- SEO meta generation (title, description, keywords)
- Preload gallery and catalog items
- Integrate Google Map pin and click-to-call / WhatsApp
- Enable optional chat or CTA buttons

7️⃣ **Extras**
- Include sample theme templates (modern, minimal, classic)
- Include example form UI for each slot section
- Include instructions for adding new themes in the future
- Optionally, generate PDF export of website preview for the vendor
